# REDMINE DASHBOARD (Template for development)

* API tunneling (API proxy)
* handlebars UI pages


### Run Server

```shell
$ REDMINE_HOST=http://redmine_url
$ REDMINE_APIKEY=API_KEY
$ npm start
```

> You can find API KEY at the left side on http://redmine_url/my/account
